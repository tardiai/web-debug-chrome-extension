## 🔧 安装方法

### 方法1：手动安装（推荐）

1. 下载代码：`git clone https://github.com/tardiai/web-debug-chrome-extension.git`
2. 打开Chrome：`chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹

### 方法2：从Releases下载

1. 进入 [Releases](链接) 下载最新版本
2. 解压ZIP文件
3. 按方法1的步骤2-5操作

## 📖 使用方法

1. 打开任意网页（如 https://baidu.com）
2. 点击扩展图标
3. 点击"📊 提取当前页面信息"
4. 信息自动复制到剪贴板
5. 粘贴给Claude/Cursor/Copilot进行调试

## 🎯 使用场景

- **CSS调试**：样式不生效？提取信息让AI分析
- **JS错误排查**：脚本加载顺序问题？一键检测
- **SEO优化**：meta标签、OG标签完整检查
- **性能分析**：页面加载时间、资源统计
- **可访问性**：找出缺失的alt、标签问题
